## FAQ

### What is a proxy servers?
Proxy server는 사용자의 컴퓨터와 인터넷 사이에서 중개자 역할을 하는 컴퓨터입니다. 인터넷에 연결할 때 실제로는 proxy server에 연결되며, proxy server가 사용자가 이용하려는 웹사이트 또는 서비스로 연결해 줍니다. Proxy server는 보안/성능 향상은 물론 정확한 클라이언트 대면 데이터 포인트 수집 등 다양한 목적에 사용할 수 있습니다.

### Why use a proxy server?
방화벽 뒤에서 특정 웹사이트에 접근하려면 proxy server를 사용해야 할 수 있습니다. 이 서버는 기기와 인터넷 사이의 중개 역할을 수행하여, 접근이 제한되거나 부정확한 정보를 표시할 수 있는 사이트에 접속하는 데 도움을 줍니다.
